# 정규표현식

## TAG : 정규표현식 , replaceAll


### 나는 조건 하나하나 직접 로직을 구현하였는데, 자바에서는 replaceAll이라는 함수를 통해 정규표현식이 가능하였다.

### 다른 사람의 풀이를 보았을때 객체지향성을 살려 클래스화시켰다.

- 
  ### 나의 풀이 

1. 7개의 규칙에 관하여 하나씩 다 직접 구현하였다. 
2. 특히 1번규칙에서는 int형의 아스키값을 뽑았는데 그냥 charAt 과 '.' 의 조건으로 해도 됐다.
3. 마지막 문자열 제거시에는 length 체크가 필요
4. 3번 규칙이 제일 어려운 규칙이었다. 


- 
  ### 타인의 풀이


```
3번 풀이는 정규식으로 하거나 나랑 비슷하게 이전문자와 현재문자가 .. 이면 현재문자 new버퍼에 등록안하게끔하였다.
```

```
모든 규칙에 정규식이 가능하였다. replaceAll로....
```
